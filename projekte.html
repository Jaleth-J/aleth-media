Heatmap Tracker

Heatmap Tracker is an experimental web tool designed to analyze user interactions within a webpage. It visualizes click behavior in real time and stores interaction data locally in the browser.

This project demonstrates how visual analysis, UI design, and lightweight data logging can be combined to gain insights into user behavior. It's built to provide a foundation for UX research, interface testing, or development teams looking to track engagement with minimal setup.

  • 🔴 Real-time heatmap rendering of user clicks
  • 📦 Stores interaction data in localStorage
  • ⬇️ Export click logs as JSON
  • 🧪 Prototype only – no backend, easy to extend

🔬 This is a functional prototype intended to visualize and explore interaction data in a browser environment.
The focus is on concept and design potential rather than full-scale analytics infrastructure.

🎮 Hiragana Daily

🎮 Hiragana Daily – Language Learning Mini Game

Overview:
Hiragana Daily is a browser-based mini game designed to reinforce the meaning of Japanese Hiragana characters. Players match each character with its correct meaning (like "sun" or "car") and can use emoji hints to improve recall.

Technology & Focus:
This project combines web development (HTML, CSS, JS) with a UX-focused learning interface. The design emphasizes simplicity, quick feedback, and optional dark mode for comfort during evening study sessions.

  • Randomly generated Hiragana characters with romaji support
  • Meaning-based multiple-choice questions
  • 💡 Emoji hint system for visual learners
  • Dark mode toggle
  • Mobile-responsive layout

Target Audience:
Beginners in Japanese language learning who want to link characters with basic vocabulary using a lightweight and playful approach.

My Role:
I independently designed and developed the full experience, including UI layout, random logic, hint system, and feedback flow in JavaScript.

This project also reflects my personal language learning journey and serves as a UX experiment in minimalist gamified learning for non-Latin scripts.

🧩 Crunchyroll Language Filter

🧩 Firefox Plugin – Fine-Grained Language Filtering for Anime

Overview:
Crunchyroll Language Filter is a browser extension that enhances the Crunchyroll interface by allowing users to filter anime by specific audio languages—such as "German," "Japanese," or "English"—instead of just the usual "Sub" or "Dub" options.

Technology & Challenges:
Built using the Firefox WebExtension API, JavaScript, and dynamic DOM parsing, the plugin relies on cross-page communication via runtime.sendMessage(). Major technical challenges included handling CORS restrictions and dealing with Crunchyroll’s single-page application (SPA) behavior.

  • Reads available audio languages directly from the DOM
  • Uses background.js to relay data between pages
  • Planned popup interface for selecting target languages
  • Early prototype currently blocks series pages by default
  • Intended for submission to Crunchyroll

Motivation:
I created this plugin out of a deep interest in browser extensions—especially how they can improve existing websites without needing access to their backend.

My Role:
I fully designed and developed the plugin architecture, messaging system, and core logic to handle CORS-safe communication and asynchronous data flows.

This plugin is also part of my personal learning journey in browser automation and DOM-level enhancement for user experience design.
